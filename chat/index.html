Normal Commands:
- /user username
Change your username
- /clear
Clears message history
- /owner
Grant ownership
- /admin
Grant administrator

Operator Commands:
- /rank rank
Set yourself a rank
- /ban username
Ban a user

Owner Commands:
- /invis
Go invisible
- /op
Grant operator
- /deop
Take back operator
- /reset
Resets localstorage
